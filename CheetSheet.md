# Cheet Sheet Serverless :memo:

 :construction: Create a new Serverless project.
```cmd
serverless
```

:building_construction: Send code to AWS
```
serverless 	deploy --verbose
```
:building_construction: Send code to AWS (Short mode)
```
sls 	deploy --verbose
```
